# Bricks
game

Control the ball to break some bricks !!!!

Control:
left : q
right: d
space: start the game
r    : restart the game

Difficulty speed:
1(touchpad): easy
2          : medium
3          : hard


Made with babylon.js

Copy the function createScene int the playground babylon.js or in your code !!!

https://playground.babylonjs.com/
